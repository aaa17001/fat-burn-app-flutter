### 缺点分析

#### 1. ¥99 可能被认为"贵" 💸

**问题：**
```
部分用户心理:
"一个 App 要 99 块？太贵了"
"订阅才 9.9，买断要 99，差 10 倍"
"再看看别的..."

- 视频会员：¥198/年

```

**应对策略：**
```
1. 强调价值
   "每天不到 3 毛钱，用一年"
   
2. 限时优惠
   "首发价¥69，原价¥99"
   
3. 对比锚定
   "私教课¥200/节，App¥99 终身"
   
4. 退款保证
   "7 天无理由退款"
```

#### 2. 订阅版可能无人问津 📉

**风险：**
```
用户计算:
"¥9.9 × 10 个月 = ¥99"
"不如直接买断"

结果:
订阅版转化率极低（1-2%）
大部分人选终身版

问题:
- 持续收入少
- 第 2 年无新收入

**数据预测：**
```
选择分布:
免费版：55%
订阅版：5%  ← 很低
终身版：40%  ← 很高

订阅版成摆设？
```

**应对策略：**
```

- 订阅版：¥19.9/月（提高）
- 终身版：¥199（提高）
- 拉开差距

方案 B: 订阅专属功能
- 云端同步
- 多设备
- 专属计划
- 只有订阅能用

方案 C: 首年优惠
- 订阅首年¥59
- 第 2 年恢复¥9.9/月
- 降低决策门槛
```

---

#### 3. 终身版用户后续成本 🔄

**问题：**
```
¥99 买断后:
- 服务器成本你承担
- Bug 修复免费
- 功能更新免费
- 可能用 3-5 年

¥99 - ¥7.2 = ¥91.8（仍然很赚）
