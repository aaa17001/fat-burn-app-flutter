## 🚀 使用指南

### 开始新对话

1. **打开 Google AI**
   ```
   https://gemini.google.com
   ```

2. **创建新对话**
   ```
   对话名称：[主题]
   ```

3. **复制提示词**
   ```
   从 Articles-Summary-and-Google-AI-Plan.md 复制提示词
   ```

4. **开始讨论**
   ```
   粘贴提示词，开始讨论
   ```

5. **保存记录**
   ```
   复制到 docs/google-ai-discussions/[序号]-[主题]-Discussion.md
   更新本索引文件
   ```

### 查看历史对话

1. **打开本索引文件**
   ```
   docs/google-ai-discussions/README.md
   ```

2. **查看对话统计表格**
   ```
   找到已完成的对话
   ```

3. **点击链接查看**
   ```
   点击对应对话记录链接
   ```
