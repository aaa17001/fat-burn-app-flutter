# Google AI 协作指南

## 💡 核心理念

**让合适的 AI 做合适的事**

| 任务类型 | 使用工具 | 原因 | 成本 |
|---------|---------|------|------|
| **项目规划** | Google AI (Gemini) | 免费、强大、适合讨论 | 💰 免费 |
| **结构分析** | Google AI (Gemini) | 免费、快速、适合头脑风暴 | 💰 免费 |
| **方案讨论** | Google AI (Gemini) | 免费、可以多轮对话 | 💰 免费 |
| **代码开发** | OpenCode | 专业、可执行、可测试 | 💰 付费 |
| **测试部署** | OpenCode | 专业、可执行 CI/CD | 💰 付费 |
| **数据分析** | OpenCode | 可访问本地数据 | 💰 付费 |

**核心策略：**
> **前期规划用 Google AI（免费） + 后期开发用 OpenCode（付费）**

---

## 🎯 工作流程

### 阶段 1: 项目立项（Google AI）

```
1. 打开 https://gemini.google.com
2. 创建新对话
3. 描述项目想法
4. 讨论可行性、风险、优化方案
5. 保存讨论结果到 docs/Google-AI-Discussion.md
```

**示例提示词：**
```
我正在开发一个 [项目类型]，核心想法是：
- [核心功能 1]
- [核心功能 2]
- [目标用户]

我想和你讨论：
1. 项目可行性
2. 潜在风险
3. 技术选型建议
4. 商业模式
5. 时间线规划

请给我详细的分析和建议。
```

**预计耗时：** 30-60 分钟  
**成本：** 💰 免费

---

### 阶段 2: 结构分析（Google AI）

```
1. 将项目想法告诉 Google AI
2. 请求系统性的结构分析
3. 讨论各个模块的职责
4. 确定接口和依赖关系
5. 保存分析结果到 docs/Structure_Analysis.md
```

**示例提示词：**
```
基于我们讨论的项目，请帮我：
1. 分析系统架构
2. 划分功能模块
3. 定义模块间接口
4. 推荐技术栈
5. 评估开发难度

请给出详细的结构分析报告。
```

**预计耗时：** 30-60 分钟  
**成本：** 💰 免费

---

### 阶段 3: 项目规划（Google AI）

```
1. 请求 Google AI 制定项目计划
2. 讨论时间线和里程碑
3. 确定优先级
4. 评估资源需求
5. 保存规划到 docs/Project_Plan.md
```

**示例提示词：**
```
请帮我制定详细的项目计划：
1. 分阶段规划（周为单位）
2. 每个阶段的任务清单
3. 里程碑和验收标准
4. 风险评估和应对
5. 资源需求估算

请给出可执行的项目计划。
```

**预计耗时：** 30-60 分钟  
**成本：** 💰 免费

---

### 阶段 4: 开发实施（OpenCode）

```
1. 将 Google AI 的规划导入项目
2. 启动 OpenCode
3. 执行具体开发任务
4. 测试和迭代
5. 部署发布
```

**提示词：**
```
读取 @docs/Google-AI-Discussion.md
读取 @docs/Structure_Analysis.md
读取 @docs/Project_Plan.md

基于以上规划，实现 [具体功能]。
```

**预计耗时：** 根据项目规模  
**成本：** 💰 付费（Token 计费）

## 📊 成本对比

### 方案 A: 全部用 OpenCode

| 阶段 | Token 用量 | 成本 |
|------|----------|------|
| 项目规划 | ~50,000 | ¥5-10 |
| 结构分析 | ~30,000 | ¥3-6 |
| 项目规划 | ~40,000 | ¥4-8 |
| 代码开发 | ~500,000 | ¥50-100 |
| **总计** | **~620,000** | **¥62-124** |

### 方案 B: Google AI + OpenCode

| 阶段 | Token 用量 | 成本 |
|------|----------|------|
| 项目规划 (Google AI) | - | 💰 免费 |
| 结构分析 (Google AI) | - | 💰 免费 |
| 项目规划 (Google AI) | - | 💰 免费 |
| 代码开发 (OpenCode) | ~500,000 | ¥50-100 |
| **总计** | **~500,000** | **¥50-100** |

### 节省效果

| 指标 | 方案 A | 方案 B | 节省 |
|------|-------|-------|------|
| Token 用量 | 620,000 | 500,000 | -19% |
| 成本 | ¥62-124 | ¥50-100 | -19% |
| 前期成本 | ¥12-24 | 💰 免费 | -100% |

**每个项目节省：¥12-24（前期规划成本）**

- ✅ 前期规划 100% 免费

- ✅ 风险更可控
