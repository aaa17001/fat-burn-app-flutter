# 产品需求文档 (PRD) v1.0

## 📋 产品概述

**产品名称：** 燃脂助手 FatBurn Coach  
**产品定位：** 科学减脂运动计算器  
**目标用户：** 需要科学减脂的人群  
**核心价值：** 基于权威研究，提供准确的脂肪燃烧数据

---

## 🎯 产品目标

### 核心目标

1. **科学依据** - 所有数据来自权威论文
2. **数据可验证** - 每个数据都有来源可查
3. **用户体验** - 简洁易用的界面
4. **功能完整** - 运动推荐、计时、追踪、统计
5. **商业模式** - 订阅制 + 数字产品

### 成功指标

| 下载量 | 1000+ 首月 | App Store/Google Play |

## 📱 功能需求

### P0 功能（必须）

#### 1. 运动推荐

**需求描述：**
根据用户 BMI、关节状况、目标，推荐最适合的运动

**功能点：**
- [ ] 输入用户信息（身高、体重、年龄、关节状况）
- [ ] 计算 BMI
- [ ] 推荐 3-5 种运动
- [ ] 显示每种运动的脂肪燃烧数据
- [ ] 显示科学依据来源

**验收标准：**
- 推荐逻辑正确
- 数据显示准确
- 来源可查可验证

**优先级：** P0  
**预计工时：** 3 天

---

#### 2. 运动计时

**需求描述：**
实时计时，显示消耗卡路里和脂肪燃烧量

**功能点：**
- [ ] 开始/停止/重置按钮
- [ ] 实时显示时长 (MM:SS)
- [ ] 实时计算消耗卡路里
- [ ] 实时计算脂肪燃烧量
- [ ] 显示当前心率区间
- [ ] 运动完成保存记录

**验收标准：**
- 计时准确
- 卡路里计算准确
- UI 流畅
- 数据保存成功

**优先级：** P0  
**预计工时：** 4 天

---

#### 3. 基础统计

**需求描述：**
展示运动历史和消耗统计

**功能点：**
- [ ] 显示今日数据（时长、卡路里、脂肪）
- [ ] 显示本周数据
- [ ] 显示本月数据
- [ ] 显示历史趋势图
- [ ] 显示总消耗

**验收标准：**
- 数据准确
- 图表清晰
- 加载快速

**优先级：** P0  
**预计工时：** 3 天

---

## 📊 数据需求

### 科学数据

所有数据必须来自权威研究：

| 数据 | 来源 | 验证状态 |
|------|------|---------|
| MET 值 | PubMed 论文 | ✅ 已验证 |
| 脂肪氧化率 | Achten 2004 | ✅ 已验证 |
| FatMax 心率区间 | Romijn 1993 | ✅ 已验证 |
| 坡度影响 | Swain 1997 | ✅ 已验证 |

### 用户数据

**本地存储：**
- 用户信息
- 运动记录
- 体重数据
- 设置

**不上传云端：**
- 所有数据本地存储
- 保护用户隐私

## 🔒 隐私和安全

```

- 不追踪用户行为

```

### 权限管理

**必要权限：**
- 活动识别（计步）
