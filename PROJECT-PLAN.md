# 燃脂助手 FatBurn Coach - Multi-Agent 项目规划

## 📋 项目概述

**项目名称：** 燃脂助手 FatBurn Coach  
**项目类型：** 科学减脂运动计算器 App  
**目标平台：** iOS + Android + Web  
**开发模式：** Multi-Agent 协作 + OpenCode 专业开发  
**质量标准：** 所有数据可查可验证，信息来源权威可靠

---

## 🎯 项目目标

### 核心目标

1. **科学依据** - 基于权威研究和论文
2. **数据可验证** - 所有数据都有来源可查
3. **用户体验** - 简洁易用的界面
4. **功能完整** - 运动推荐、计时、追踪、统计
5. **商业模式** - 订阅制 + 数字产品

### 成功指标

| 指标 | 目标值 | 验证方式 |
|------|--------|---------|
| 用户数 | 1000+ 首月 | App Store 下载量 |
| 付费转化 | 5%+ | 收入/下载量 |
| 用户留存 | 40%+ (7 天) | 后台统计 |
| 科学准确度 | 100% 可验证 | 来源检查 |

## 📚 第一阶段：研究阶段（Week 1）

### 目标
查找权威的减脂运动研究数据和论文，确保所有功能有科学依据

### 负责智能体：Research Agent

#### Research Agent 职责

```yaml
角色：研究分析师
技能:
  - 学术论文检索
  - 数据验证
  - 来源核实
  - 信息整理

任务:
  - 检索 PubMed、Google Scholar 等权威数据库
  - 查找减脂运动相关研究论文
  - 验证脂肪燃烧比例数据
  - 整理权威数据来源
  - 生成研究报告

交付成果:
  - 研究数据汇总表
  - 论文引用列表
  - 数据来源文档
  - 科学依据报告
```

#### 具体任务清单

**任务 1.1: 脂肪燃烧比例研究**
```
搜索关键词:
- "fat oxidation rate exercise intensity"
- "fat burning heart rate zone"
- "FatMax exercise"

目标:
- 找到不同运动强度下的脂肪氧化率数据
- 验证快走 70%、爬坡走 60% 等数据的准确性
- 记录论文来源和实验方法

交付:
- 脂肪燃烧数据表（含来源）
- 相关论文 PDF
- 数据验证报告
```

**任务 1.2: 运动类型效果对比**
```
搜索关键词:
- "walking vs running fat loss"
- "HIIT vs steady state cardio"
- "exercise type fat oxidation"

目标:
- 对比不同运动类型的减脂效果
- 验证各运动 MET 值
- 整理权威推荐

交付:
- 运动效果对比表
- MET 值来源文档
- 推荐运动列表
```

**任务 1.3: NEAT 研究**
```
搜索关键词:
- "NEAT weight loss"
- "non-exercise activity thermogenesis"
- "daily steps weight loss"

目标:
- 验证 8000-10000 步的科学依据
- 查找 NEAT 与减脂关系研究
- 整理权威建议

交付:
- NEAT 研究报告
- 步数建议来源
- 相关论文列表
```

**任务 1.4: 心率区间研究**
```
搜索关键词:
- "fat burning heart rate zone"
- "maximum heart rate calculation"
- "exercise intensity zones"

目标:
- 验证 FatMax 心率区间计算方法
- 查找不同年龄心率区间数据
- 整理权威公式

交付:
- 心率区间计算公式
- 年龄对照表
- 来源文档
```

### 研究阶段验收标准

- ✅ 所有数据都有论文/研究支持
- ✅ 来源可查（PubMed ID/DOI）
- ✅ 数据汇总表完整
- ✅ 研究报告通过审核
